# Gravity Flight Simulator

A lightweight Three.js flight sandbox that runs in-browser. Fly a stylized plane by default, or switch to a flying broomstick or X-Wing inspired ride using the in-app selector.

## Features
- Plane, broomstick, and X-Wing style craft options with instant switching.
- Keyboard controls on desktop (W/S throttle, arrow keys for pitch/roll) and touch controls on mobile.
- Dynamic camera chase view, rolling hills, and drifting clouds for depth.
